# Render + iPad Video Call Pro

Files:
- index.html
- server.js
- package.json

## Local run
npm install
npm start

## Render deploy
Build command:
npm install

Start command:
node server.js

## Notes
- Use the Render HTTPS URL on all devices.
- On iPad iOS 12, use Safari only.
- This is optimized for old Safari, but old iPad WebRTC support is still limited.
